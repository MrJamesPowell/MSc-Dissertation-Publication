# MSc-Dissertation-Publication
Repository of code used for the authors dissertation for a Master of Science in Mechanical Engineering, titled "Design, Manufacture and Evaluation of a Heated Blanket for Treatment of Hypothermia".
This repository also contains code used for the additional work for publication of the dissertation material.
