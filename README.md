# MSc-Dissertation-Publication
Repository of code used for the authors dissertation for a Master of Science in Mechanical Engineering, titled "Design, Manufacture and Evaluation of a Heated Blanket for Treatment of Hypothermia".<br>
This repository also contains code used for the additional work for publication of the dissertation material.<br>

---

## Component list
### Resistors
R1 = 4.7K<br>
R2 = 1K<br>
R3 = 10K<br>
_Note: All resistors used were 1/8 W metal film_<br>
